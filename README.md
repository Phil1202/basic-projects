# 1. Color Flipper

For my first JavaScript basic project, I decided to code a **"Color Flipper"** website, a simple background color changer triggered by a "click me" button. The project is based on the tutorial by [John Smilga](https://www.youtube.com/watch?v=3PHXvlpOkf4&t=421s) .
While coding, I concerned with several key concepts:  

- ***arrays***: List-like objects whose prototype has methods to perform traversal and mutation operations.

- ***document.getElementById()***: A method that returns an Element object representing the element whose id property matches the specified.

- ***document.querySelector()***: A method that returns the first element within the document that is a descendant of node matches selections.

- ***addEventListener()***: A method of the Event Target interface that sets up a function that will be called whenever the specified event delivered to the target.

- ***document.body.style.backgroundColor***: The style property is used to get or set a specific style of an element using different CSS properties. The backgoundColor property sets or returns the background color of an element. 

- ***Math.floor()***: A function that returns the largest integer less than or equal to a given number.

- ***Math.random()***: A function that returns a pseudorandom number between 0 and 1 (but not 1).

- ***array.length***: The length property of an object which is an instance of type Array sets or returns the number of elements in that array. 

- ***for loop***: A for loop repeats until a specified condition evaluates to false. 
         
		   	for([initialExpression]; [conditionExpression]; [incrementExpression]}
